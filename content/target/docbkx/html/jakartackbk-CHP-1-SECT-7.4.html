1.7.4. See Also

compareTo( ) methods provide the natural sorting order for a set of objects, and they are frequently used when sorting a collection of JavaBeans©. If you are trying to sort a collection of beans, you are better off using the BeanComparator, which is described in Recipe 3.10.

Instead of capturing comparison logic in a compareTo() method, consider using a Comparator object. The Jakarta Commons Collections project contains a number of supplements to the Comparator interface, such as utilities to reverse and chain comparators. Comparator utilities are discussed in Chapter 4.
