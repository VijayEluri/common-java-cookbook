11.8.3. Discussion

This recipe sets the request body of an HTTP POST directly by passing a File object to method.setRequestBody(). In addition to accepting a File object, setRequestBody( ) can accept an InputStream or a String. Any time a request body is populated, the Content-Length header must be set to reflect the size of the request body by calling method.setRequestContentLength( ). The previous recipe sent parameters in an HTTP POST request body by calling setParameter( ) and addParameter(), and the Content-Length and Content-Type headers are automatically populated when the method is executed. In this example, the Content-Type header is not sent with the request as the content can be any arbitrary textual or binary data.
