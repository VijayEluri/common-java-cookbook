3. What You'll Need

To experiment with the various libraries introduced in this book, you will need the J2SE 1.4 SDK, which can be obtained from http://www.javasoft.com. The examples in this book were developed in an IDE named Eclipse, which can be downloaded from the Eclipse project site at http://www.eclipse.org. Each chapter contains instructions for downloading the various Jakarta Commons components; to download any component from Jakarta Commons, go to the Jakarta Commons page at http://jakarta.apache.org/commons and click on "Binaries" under "Download" in the left menu.
